About the Project

ML Stack

Dataset: Branch-level deposits with demographics, geography (state, district), channel mix, product type, and temporal attributes (year/quarter).

Objective: Predict deposit amounts and group customers into actionable clusters for marketing and branch optimization.

Techniques: Regression (Linear, Polynomial, Random Forest, Gradient Boosting, KNN, SVR, CNN) and K-Means clustering.

Evaluation: RÂ², RMSE, MAE, cross-validation mean score to ensure generalization.

Tools: Python, Flask, scikit-learn, pandas, matplotlib, seaborn, Chart.js, TailwindCSS.

APIs: /predict for inference, /get_model_results for metrics, /get_clusters for segmentation.

CNN Regression: 1D convolution on temporal features for smoothing seasonality; summary available via API and on results page.

Deliverables: Downloadable reports for model comparison and cluster analysis with ready-to-embed PNG charts.
